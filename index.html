Projects

Mechanistic Interpretability of Open Source LLMs: reproduction of On the Biology of a Large Language Model

Investigation of mechanisms can be found in an open model, Qwen3-4B-Instruct. DAMPT, University of Cambridge.

Electronic Nose: Machine Learning for Gas Sensing

Developed machine learning models for gas sensing detection using semiconductor sensor arrays. Research focused on pattern recognition, signal processing, and neural network interpretability.

Subprojects:

  • Feature Analysis in Physics Data: Interpretability analysis using different methods for feature selection and extraction. Applied Weight Analysis and Deep Taylor Decomposition of neural networks to understand which features contribute most to gas sensing predictions.
  • Calibration of Neural Networks: Investigated calibration techniques for neural network classifiers applied to sensor data, based on Guo et al. (2017) but adapted to gas sensing applications. Developed methods to improve probability estimates and model reliability under domain shift and sensor drift conditions.

PWD Studies: Pulse-Wave Doppler Analysis

Developed machine learning models for automated analysis of Pulse-Wave Doppler signals in cardiovascular diagnostics. Implemented signal preprocessing pipelines and classification algorithms to identify pathological patterns in arterial blood flow.
